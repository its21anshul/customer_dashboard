# 📊 Customer Behavior Dashboard

A professional and interactive **Power BI Dashboard** built to analyze customer purchasing behavior, sales trends, and business performance. This dashboard provides actionable insights into customer demographics, subscription status, revenue distribution, and sales across various categories.

---

## 📌 Project Overview

The **Customer Behavior Dashboard** helps businesses understand customer purchasing patterns and sales performance through interactive visualizations. It enables users to filter data by customer demographics, subscription status, product categories, and shipping methods to make informed business decisions.

---

## 🚀 Features

- 📈 Interactive KPI Cards
- 👥 Customer Demographics Analysis
- 💰 Revenue Analysis
- 🛒 Sales Performance Tracking
- ⭐ Customer Review Ratings
- 📦 Product Category Analysis
- 🚚 Shipping Type Filters
- 🎯 Dynamic Slicers & Filters
- 📊 Business Intelligence Dashboard
